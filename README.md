# Visual Arts Practice

This repository contains my visual arts and design practice work created while learning the fundamentals of design.

## Purpose

The goal of this repository is to improve my understanding of core design principles through hands-on practice and visual experimentation.

## Topics Covered

- Principles of Design
- Balance
- Contrast
- Emphasis
- Movement
- Rhythm
- Unity
- Proportion
- Visual Composition
- Layout Practice
- Sketch Studies (scanned copies)

## Content

This repo includes:
- Scanned sketches and practice sheets
- Design principle exercises
- Visual studies and experiments

## Note

This is a learning repository. The work here is not final or professional-level output, but part of my learning journey in visual design.
