## Elements of Art

(These are different from principles — these are the building blocks.)

-> Line
-> Shape
-> Form
-> Color
-> Value
-> Texture
-> Space

Simply:
Elements = ingredients
Principles = how you arrange/use the ingredients

Like:
flour, sugar, butter = elements
recipe technique = principles


## Principles of Design
-> Balance
-> Contrast
-> Emphasis
-> Movement
-> Pattern
-> Rhythm
-> Unity
-> Variety
-> Proportion
-> Repetition
-> Alignment
-> Hierarchy
-> White Space (Negative Space)
-> Scale
